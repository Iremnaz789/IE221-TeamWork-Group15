# IE221 – Team Work 3
## Experimental Verification of SLLN and CLT

This project experimentally verifies the Strong Law of Large Numbers (SLLN)
and the Central Limit Theorem (CLT) using simulation methods.
Additionally, a Monte Carlo method is used for estimating π.
